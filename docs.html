Ein einfaches Beispiel für einen HTML-Code. Perfekt für grundlegende Erklärungen.

</body>

Code mit Erklärung

Ein Code-Beispiel mit einer detaillierten Erklärung, wie es funktioniert. 


<div class="flex justify-center">
    <button class="bg-blue-500 text-white px-4 py-2 rounded">
        Klick mich
    </button>
</div>

Dieser Code erstellt einen zentrierten Button mit blauem Hintergrund und weißem Text.
